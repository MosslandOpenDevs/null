# 용어 사전

| 용어 | 정의 |
|---|---|
| **아쿠아리움 모드 (Aquarium Mode)** | 세컨드 스크린이나 스크린세이버용 크롬리스 풀스크린 코스모그래프 디스플레이. 터치 제스처로 모바일 최적화. |
| **아키텍트 (Architect)** | 제네시스 노드의 세계 구축 AI 역할. 세계 설정과 초기 에이전트 페르소나 생성을 담당. |
| **아카이브 (The Archive)** | 북마크 및 내보내기 시스템. 뷰 상태 북마크(`B`키), 클립 녹화(`R`키), 스토리 아크 내보내기를 지원. |
| **정전 (Canon)** | 다중 에이전트 합의를 통해 확립된 사실. 위키에서 검증되고 불변. 오라클 패널에서 솔리드 텍스트로 렌더링. |
| **카오스 조커 (Chaos Joker)** | 반대 의견을 주입하고 합의를 방해하도록 설계된 에이전트 역할. |
| **코스모그래프 (Cosmograph)** | 풀스크린 WebGL 공간 시각화 — 옴니스코프의 핵심 뷰. 에이전트가 발광 노드로 나타나며 관계 필라멘트로 연결됨. Three.js + 커스텀 셰이더로 렌더링. |
| **신의 개입 (Divine Intervention)** | 인간이 시뮬레이션에 영향을 미치는 공간 제스처 기반 시스템 (디렉터 모드 대체). 이벤트 드롭, 위스퍼 모드, 시드 봄 포함. |
| **엔트로피 주입 (Entropy Injection)** | 합의 함정을 방지하기 위해 랜덤 또는 파괴적 이벤트를 의도적으로 도입하는 것. |
| **에포크 (Epoch)** | 시뮬레이션의 이산 시간 단위로, 문명 발전의 중요한 단계를 나타냄. 타임라인 리본에서 색상으로 구분. |
| **이벤트 드롭 (Event Drop)** | 신의 개입 메커닉: 이벤트 토큰(위기, 발견, 역병, 지도자 교체)을 코스모그래프에 드래그하여 다양한 범위의 이벤트를 트리거. |
| **제네시스 노드 (Genesis Node)** | 시드 프롬프트로부터 새로운 시뮬레이션 세계를 생성하는 엔진. |
| **헤럴드 (The Herald)** | 시뮬레이션 이벤트의 내러티브 산문 요약을 전달하는 AI 기반 알림 시스템. "Catch me up" AI 브리핑 포함. |
| **하이브 마인드 (Hive Mind)** | 시뮬레이션의 살아있는 위키를 유지하는 집단 지성 엔진. |
| **전설 (Legends)** | 위키에서 미검증 또는 논쟁 중인 주장. 합의를 통해 정전으로 승격될 수 있음. 오라클 패널에서 쉬머/불확실 효과로 렌더링. |
| **라이브러리언 (Librarian)** | 위키 유지보수, 요약, 지식 그래프 업데이트를 담당하는 에이전트 역할. |
| **관찰자 (Observer)** | 직접 상호작용 없이 시뮬레이션을 관찰하는 인간 사용자. |
| **옴니스코프 (The Omniscope)** | NULL의 완전한 관찰 인터페이스. 코스모그래프, 타임라인 리본, 오라클 패널, 헤럴드, 아카이브로 구성. 기존 갓뷰 대시보드 / 트리니티 뷰를 대체. |
| **오라클 패널 (Oracle Panel)** | 에이전트 프로필, 관계 이력, 파벌 위키, 이벤트 상세를 표시하는 컨텍스트 트리거 슬라이드인 패널 (프로스트 글라스). 고정 3패널 레이아웃을 대체. |
| **시드 봄 (Seed Bomb)** | 신의 개입 메커닉: 지역에 토픽을 드롭하여 에이전트들이 해당 주제를 토론하게 하고, 파문을 시각화. |
| **시드 프롬프트 (Seed Prompt)** | 시뮬레이션의 매개변수(시대, 문명, 갈등 등)를 정의하는 초기 입력. |
| **시맨틱 줌 (Semantic Zoom)** | 코스모그래프의 5단계 연속 줌 시스템: 우주(L1) → 파벌(L2) → 클러스터(L3) → 대화(L4) → 에이전트(L5). |
| **스웜 (Swarm)** | 집단적으로 행동하는 에이전트 그룹, 주로 파벌이나 동맹 내에서. |
| **시간 왜곡 (Time Dilation)** | 실제 시간과 시뮬레이션 시간 사이의 비율을 제어하는 엔진. |
| **타임라인 리본 (Timeline Ribbon)** | 옴니스코프 하단의 수평 스트립으로, 에포크 색상 코딩과 시간 스크러빙 제어로 시뮬레이션 역사를 표시. |
| **위스퍼 모드 (Whisper Mode)** | 신의 개입 메커닉: 에이전트를 우클릭하여 텍스트를 다음 추론 사이클의 "내면의 목소리"로 주입. |
| **세계 설정 (World Config)** | 모든 시뮬레이션 매개변수를 정의하는 제네시스 노드의 구조화된 출력. |
| **북마크 (Bookmark)** | 관심 있는 엔티티를 저장하고 내보내기 위한 사용자 생성 컬렉션 항목. |
| **이머전트 택소노미 (Emergent Taxonomy)** | 임베딩 클러스터에서 자동 생성된 바텀업 카테고리 트리. 인간 분류 없이 LLM이 노드를 계층적으로 레이블링. |
| **엔티티 멘션 (Entity Mention)** | 자연어 텍스트 내에서 에이전트, 위키 페이지, 파벌에 대한 자동 감지된 참조. 양방향 링크를 생성. |
| **시맨틱 네이버 (Semantic Neighbor)** | pgvector 코사인 거리로 발견된, 다른 엔티티와 높은 임베딩 유사도를 가진 엔티티. |
| **시맨틱 세디먼트 (Semantic Sediment)** | 에이전트가 생성한 콘텐츠가 자동으로 레이어링, 링크, 분류되는 데이터 조직 패러다임. |
| **스트라텀 (Stratum)** | 단일 에포크에 해당하는 시간 지층. 출현한 개념, 퇴조한 개념, 지배적 주제를 기록. 복수형: 스트라타(strata). |
| **택소노미 노드 (Taxonomy Node)** | 이머전트 택소노미 트리의 노드. 리프(단일 클러스터) 또는 부모(병합된 클러스터)일 수 있음. |
| **학습 데이터 내보내기 (Training Export)** | LLM 파인튜닝 형식의 월드 데이터 내보내기: ChatML, Alpaca, 또는 ShareGPT. |
