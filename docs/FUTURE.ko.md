# 미래 확장

## 1. 노드 간 전쟁

여러 제네시스 노드를 연결하여 경쟁하는 문명을 생성합니다. 노드 간에 무역, 스파이, 협상, 전쟁이 가능합니다.

**개념:** 각 노드는 독립적인 시뮬레이션입니다. 메타 레이어가 노드 간 외교와 갈등을 관리합니다. 다른 노드의 에이전트들은 공유 컨텍스트가 없어 — 상호작용을 통해 이해를 구축해야 합니다.

## 2. 멀티모달 진화

텍스트를 넘어 확장합니다:
- **이미지 생성** — 에이전트가 시각 예술, 지도, 건축 설계도 생성
- **제너레이티브 오디오** — 시뮬레이션 상태에 따라 변화하는 Tone.js 앰비언트 사운드스케이프 (옴니스코프의 선택적 사운드 디자인으로 이미 통합됨; 향후 에이전트 생성 음악, 연설, 프로파간다 방송으로 확장)
- **비디오** — 아카이브 시스템의 클립 녹화가 코스모그래프의 WebM 캡처를 내보냄; 향후 AI 생성 시네마틱 요약으로 확장
- **공간 시각화** — 코스모그래프의 Three.js 렌더러가 시뮬레이션 공간의 3D/VR 확장을 위한 기반 제공

## 3. 서비스형 NULL (NaaS)

NULL을 호스팅 플랫폼으로 패키징:
- 사용자가 웹 인터페이스를 통해 시드 프롬프트 제출
- 설정 가능한 매개변수로 클라우드에서 시뮬레이션 실행
- 결과를 다운로드 가능한 데이터셋과 호스팅 위키로 전달
- 에이전트 수와 시뮬레이션 기간에 따른 구독 티어

## 4. 에이전트 경제

에이전트가 자원, 정보, 영향력을 거래하는 내부 경제 도입:
- 정보 가치에 연동된 **통화 시스템**
- 지식, 동맹, 서비스를 거래하는 **시장**
- 에이전트 상호작용의 창발적 속성으로서의 **경제 모델링**

## 5. 교육 응용

NULL의 교육적 활용:
- **역사 교육** — 역사적 시나리오를 시뮬레이션하고 대안적 결과를 관찰
- **정치학** — 거버넌스 시스템과 정책 영향 모델링
- **윤리 교육** — 에이전트 담론을 통한 도덕적 딜레마 탐구
- **창작 글쓰기** — 세계 구축 자료와 서사 구조 생성

## 6. 연구 플랫폼

연구를 위한 API와 도구 제공:
- **사회 역학 연구** — 통제된 환경에서 창발적 행동 연구
- **LLM 행동 분석** — 다양한 모델이 사회적 맥락에서 어떻게 행동하는지 관찰
- **합의 메커니즘 연구** — 대규모로 투표 및 합의 프로토콜 테스트

## 7. 옴니스코프 확장

관찰 레이어의 향후 개선:
- **VR/AR 모드** — 3D 공간에서 코스모그래프 내부로 진입
- **다중 관찰자** — 여러 인간이 공유/개인 북마크로 동일 시뮬레이션을 동시 관찰
- **AI 내레이터** — 시뮬레이션 이벤트의 실시간 음성 내레이션을 제공하는 전용 LLM
- **절차적 음악** — 문명 상태에 따라 진화하는 풀 제너레이티브 사운드트랙
