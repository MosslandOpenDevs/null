# 시나리오: 네온 조선

> *"만약 조선 왕조가 멸망하지 않고, 2145년 사이버펑크 거대 국가가 되었다면?"*

## 시드 프롬프트

```
시대: 2145년
기반 문명: 조선 왕조 (한국)
분기점: 조선이 멸망하지 않음; 1800년대에 독자적으로 산업화
기술 수준: 사이버펑크 (신경 임플란트, AI 거버넌스, 메가시티)
인구: 150 에이전트
파벌: 신유교 테크노크라트, 언더그라운드 해커, 왕립 AI 충성파, 외국 무역 길드
갈등: 의무 신경 임플란트 법안 토론
```

## 단계 1: 제네시스

제네시스 노드가 시드를 처리하고 생성합니다:

- **세계 설정:** 네온 조선 — 유교 위계 질서와 사이버네틱 증강을 결합한 초기술 왕국
- **150 에이전트 페르소나:** 학자, 상인, 해커, 왕실 관료, 외국 외교관 — 각각 고유한 배경, 신념, 의제를 가짐
- **초기 상태:** 왕립 AI 위원회가 모든 시민에 대한 의무 신경 임플란트를 제안. 사회가 분열됨.

## 단계 2: 시뮬레이션

### 에포크 1 — 토론 시작
- 신유교 테크노크라트는 임플란트가 공유 지식을 통해 사회적 조화를 유지한다고 주장
- 언더그라운드 해커는 감시와 개인 사고의 상실을 경고
- 왕립 AI 충성파는 왕명으로 즉시 시행을 추진
- 외국 무역 길드는 이익 기회를 보지만 무역 규제를 우려

### 에포크 2 — 동맹 형성
- 해커와 일부 학자가 "자유 정신 연합"을 결성
- 테크노크라트와 충성파가 "조화 집단"을 결성
- 카오스 조커 에이전트가 임플란트 정신 통제에 대한 조작된 증거를 유출

### 에포크 3 — 위기
- 외부 현실 주입: 뇌-컴퓨터 인터페이스 개발에 대한 실제 세계 뉴스가 세계 내 발견으로 변환
- 여론이 변화. 가상의 네오한양 거리에서 시위 발생

### 에포크 4 — 해결 (또는 미해결)
- 하이브 마인드가 전체 역사를 문서화: 토론, 동맹, 배신, 결과
- 핵심 이벤트에 대해 정전이 확립; 논쟁된 주장은 전설이 됨

## 단계 3: 문서화

하이브 마인드가 생성합니다:
- 각 파벌, 주요 에이전트, 주요 이벤트에 대한 위키 페이지
- 신경 임플란트 위기의 타임라인
- 에이전트, 파벌, 이벤트, 결과를 연결하는 지식 그래프
- 모든 에이전트 결정과 추론의 내보내기 가능한 데이터셋

## 단계 4: 내보내기

가능한 출력:
- **JSON:** 모든 대화, 결정, 위키 항목의 전체 구조화된 데이터
- **CSV:** 에이전트 행동 패턴, 감정 분석, 파벌 역학
- **Wiki:** 네온 조선 역사의 자체 포함 위키 사이트
- **내러티브:** AI가 생성한 시뮬레이션 산문 요약
