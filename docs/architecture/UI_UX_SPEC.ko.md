# UI/UX 명세 — 갓뷰 대시보드

## 개요

갓뷰 대시보드는 시뮬레이션에 대한 세 가지 동시적 관점을 제공하는 **트리니티 뷰** 레이아웃입니다.

## 레이아웃: 트리니티 뷰

```
┌─────────────────┬─────────────────────┬─────────────────┐
│                  │                     │                 │
│   카오스 스트림    │    질서 / 위키       │   분석 & 제어     │
│   (좌측 패널)     │   (중앙 패널)        │  (우측 패널)      │
│                  │                     │                 │
└─────────────────┴─────────────────────┴─────────────────┘
```

## 좌측 패널: 카오스 스트림

모든 에이전트 활동의 원시 실시간 피드.

**기능:**
- 실시간 대화 피드 (스크롤 가능, 에이전트/주제별 필터링)
- 에이전트 상태 표시기 (활성, 토론 중, 대기, 음모 중)
- 감정 색상 코딩 (녹색: 동의, 빨강: 갈등, 노랑: 중립)
- 주요 이벤트 하이라이트 마커 (동맹 형성, 배신, 발견)
- 클릭하여 전체 대화 스레드 확장

## 중앙 패널: 질서 / 위키

구조화된 지식 출력 — 하이브 마인드의 위키.

**기능:**
- 검색 가능한 위키 페이지 브라우저
- 실시간 편집 표시기 (어떤 에이전트가 무엇을 편집 중인지)
- 정전 vs. 전설 구분 (검증된 사실 vs. 논쟁 중인 주장)
- 지식 그래프 시각화 (D3.js 힘 방향 그래프)
- 문명 역사의 타임라인 뷰
- 위키 페이지 변화를 위한 Diff 뷰

## 우측 패널: 분석 & 제어

시뮬레이션 지표와 인간 제어 인터페이스.

**기능:**
- 인구 통계 (에이전트 수, 파벌 분포, 감정 분포)
- 시간 제어 (재생, 일시정지, 빨리감기, 에포크로 되감기)
- 이벤트 주입 (수동으로 이벤트 트리거: 전쟁, 역병, 발견)
- 내보내기 제어 (데이터 범위, 형식 선택, 다운로드)
- 비용 모니터 (API 사용량, 남은 예산)
- 시뮬레이션 건강 상태 (에러율, 지연시간, 큐 깊이)

## 상호작용 모델

- **관찰자 모드 (기본):** 읽기 전용. 시뮬레이션이 전개되는 것을 관찰.
- **디렉터 모드:** 이벤트 주입, 시간 조정, 매개변수 수정.
- **분석가 모드:** 데이터 내보내기, 필터링, 시각화에 집중.
